WIFI AUTOMATION SYSTEM - COMPREHENSIVE FEATURE LIST
=====================================================

PROJECT OVERVIEW
================
The WiFi Automation System is a comprehensive solution for MoonFlower Hotel that automates the entire workflow of:
1. Downloading WiFi user data from Ruckus Wireless Controller
2. Processing and converting CSV files to Excel format
3. Integrating with VBS (AbsonsItERP) application
4. Generating PDF reports
5. Sending automated email notifications

CORE FEATURES
=============

1. WEB SCRAPING MODULE
---------------------
- **Bulletproof Ruckus WiFi Scraper**: Automated login and data extraction from Ruckus Wireless Controller
- **Network Sources**: Extracts data from 4 WiFi networks:
  * EHC TV (Page 1, requires clients tab)
  * EHC-15 (Page 1, direct download)
  * Reception Hall-Mobile (Page 2, requires clients tab)
  * Reception Hall-TV (Page 2, direct download)
- **Anti-Detection**: Chrome automation with anti-bot detection measures
- **Retry Logic**: Automatic retry for failed downloads with exponential backoff
- **Dynamic Date Folders**: Creates date-specific folders (e.g., 05july, 06july)
- **File Organization**: Automatically organizes downloaded files with proper naming

2. FILE PROCESSING SYSTEM
------------------------
- **CSV Processing**: Reads and validates CSV files from multiple sources
- **Data Cleaning**: Removes duplicates, validates IP addresses, standardizes MAC addresses
- **Excel Generation**: Converts CSV data to Excel format (.xls) with specific headers:
  * A: Hostname
  * B: IP_Address
  * C: MAC_Address
  * D: Package (WLAN SSID)
  * E: AP_MAC
  * F: Upload (Data Rate up)
  * G: Download (Data Rate down)
- **Header Mapping**: Automatic mapping from Ruckus format to required Excel format
- **Encoding Support**: Handles multiple file encodings (UTF-8, Latin-1, CP1252)

3. VBS INTEGRATION
-----------------
- **AbsonsItERP Integration**: Automates VBS application for hotel management
- **Dual Path Strategy**: Primary and fallback paths for VBS application
- **Login Automation**: Automatic login with credentials (Username: "Vj")
- **Data Import**: Imports Excel files into VBS system
- **PDF Generation**: Generates PDF reports of active WiFi users
- **Error Recovery**: Robust error handling for buggy VBS application
- **Window Management**: Handles window detection and UI automation

4. EMAIL NOTIFICATION SYSTEM
---------------------------
- **Professional Email Service**: Gmail SMTP integration with TLS encryption
- **Branded Templates**: MoonFlower Hotel branded email templates
- **PDF Attachments**: Automatic PDF report attachments (50MB limit)
- **Report Summary**: Includes user count, file size, and report statistics
- **Unique Report IDs**: Generates unique report identifiers
- **Retry Mechanism**: 3 retry attempts with 60-second delays
- **Error Handling**: Comprehensive error logging and recovery

5. SCHEDULING SYSTEM
-------------------
- **Enhanced Service Runner**: Background service with crash prevention
- **Multiple Time Slots**: 
  * Morning: 09:30 AM (configurable)
  * Afternoon: 13:00 PM (1:00 PM)
  * Evening: 21:05 PM (9:05 PM) - Main production schedule
- **Windows Integration**: Automatic startup with Windows
- **Health Monitoring**: System health checks every 5 minutes
- **Auto-Restart**: Automatic service restart on crashes
- **Background Operation**: Runs even when PC is locked

6. MONITORING & LOGGING
----------------------
- **Comprehensive Logging**: Detailed logs for all operations
- **Screenshot Capture**: Debug screenshots for troubleshooting
- **Performance Metrics**: Success rates, processing times, file counts
- **Error Tracking**: Detailed error logs with stack traces
- **Health Checks**: System health monitoring and reporting
- **Status Reports**: Real-time status updates and notifications

7. CONFIGURATION MANAGEMENT
--------------------------
- **Environment Variables**: Secure credential management
- **Production Settings**: Separate production and development configurations
- **Encryption**: Credential encryption with Fernet keys
- **Configurable Paths**: Customizable directory paths
- **Network Configuration**: Flexible network source configuration
- **Timing Configuration**: Adjustable timeouts and delays

DIRECTORY STRUCTURE
==================

Primary Data Directories:
- EHC_Data/: Raw CSV files organized by date (e.g., 05july/)
- EHC_Data_Merge/: Processed Excel files by date (e.g., 05july/)
- EHC_Data_Pdf/: Generated PDF reports by date (e.g., 05july/)

System Directories:
- logs/: System logs and debug information
- downloads/: Temporary download directory
- config/: Configuration files and settings
- modules/: Core system modules
- server/: Backend API server
- src/: Frontend React application

WORKFLOW PROCESS
===============

1. **Scheduled Execution**:
   - Service runner triggers at scheduled times
   - Creates date-specific directories if not exist
   - Initializes web scraper with Chrome automation

2. **Data Collection**:
   - Logs into Ruckus Wireless Controller
   - Navigates through WiFi network pages
   - Downloads CSV files from 4 network sources
   - Organizes files in date-specific folders

3. **Data Processing**:
   - Reads all CSV files from date folder
   - Cleans and validates data
   - Removes duplicates and invalid entries
   - Converts to Excel format with proper headers

4. **VBS Integration**:
   - Launches AbsonsItERP application
   - Imports Excel data into VBS system
   - Generates PDF reports of active users
   - Saves reports to date-specific PDF folder

5. **Email Notification**:
   - Sends professional email with PDF attachment
   - Includes report summary and statistics
   - Provides unique report ID for tracking
